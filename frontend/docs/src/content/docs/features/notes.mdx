# Notes

The Notes feature in Escruta allows you to capture thoughts, insights, and key information while researching. With AI-powered assistance, your note-taking becomes more intelligent and organized.

## Smart Note-Taking

Escruta's note-taking system goes beyond simple text editing:

- **Rich Text Editor**: Format your notes with headers, lists, links, and more
- **AI Suggestions**: Get intelligent suggestions for expanding your notes
- **Linked References**: Connect notes to specific sources and other notes
- **Real-time Sync**: Your notes are automatically saved and synchronized

## Creating Notes

1. **Open a Notebook**: Navigate to the notebook where you want to add notes
2. **Click "New Note"**: Start creating a new note entry
3. **Add Content**: Use the rich text editor to write your notes
4. **Link Sources**: Reference specific sources or other notes
5. **Save**: Your note is automatically saved as you type

## Note Organization

- **Tags**: Add tags to categorize your notes
- **Folders**: Organize notes into folders within notebooks
- **Search**: Use the search function to find specific content
- **Filters**: Filter notes by date, tags, or source references

## AI-Powered Features

### Smart Suggestions
As you write, Escruta's AI can suggest:
- Related concepts to explore
- Questions to investigate further
- Connections to other notes and sources

### Content Expansion
The AI can help you:
- Elaborate on brief notes
- Provide context for complex topics
- Generate related questions for further research

## Best Practices

- **Use Descriptive Titles**: Make notes easy to identify and search
- **Link to Sources**: Always reference the source material
- **Regular Review**: Periodically review and update your notes
- **Use Tags Consistently**: Develop a consistent tagging system
- **Break Down Complex Ideas**: Split complex topics into multiple focused notes
