# Sources

Sources are the foundation of your research in Escruta. You can upload documents, add web links, and import various types of content that will be processed and made available for AI-powered analysis and note-taking.

## Supported Source Types

Escruta supports a wide variety of source materials:

- **Documents**: PDFs, Word documents, text files
- **Web Content**: URLs, articles, blog posts
- **Academic Papers**: Research papers, journals, citations
- **Media Files**: Images with text, transcripts
- **Data Files**: Spreadsheets, CSV files

## Adding Sources

### Upload Documents
1. Navigate to your notebook
2. Click "Add Source" → "Upload Document"
3. Select files from your computer
4. Wait for processing to complete
5. Review extracted content and metadata

### Add Web Links
1. Click "Add Source" → "Web Link"
2. Paste the URL of the webpage or article
3. Escruta will automatically fetch and process the content
4. Edit title and add tags if needed

### Import from Citations
1. Use "Add Source" → "Citation"
2. Paste citation information or DOI
3. Escruta will attempt to find and import the source
4. Verify the imported content

## Source Processing

When you add a source, Escruta automatically:

- **Extracts Text**: Converts documents to searchable text
- **Generates Summaries**: Creates AI-powered summaries
- **Identifies Key Topics**: Tags relevant subjects and themes
- **Creates Embeddings**: Enables semantic search capabilities

## Managing Sources

### Organization
- **Tags**: Add descriptive tags to categorize sources
- **Collections**: Group related sources together
- **Notes**: Add personal notes and annotations
- **Ratings**: Rate sources for quality and relevance

### Search and Discovery
- **Full-text Search**: Find specific quotes or concepts
- **Semantic Search**: Discover related content using AI
- **Filter Options**: Filter by type, date, tags, or ratings
- **Similar Sources**: Find related materials automatically

## AI-Powered Features

### Smart Summaries
Each source gets automatically generated summaries at different levels:
- **Quick Overview**: 2-3 sentence summary
- **Key Points**: Main arguments and findings
- **Detailed Summary**: Comprehensive analysis

### Content Analysis
The AI analyzes sources to identify:
- Main themes and topics
- Key arguments and evidence
- Relationships to other sources
- Potential research questions

## Best Practices

- **Quality Control**: Review AI-generated summaries for accuracy
- **Consistent Tagging**: Use a systematic approach to tagging
- **Regular Cleanup**: Remove duplicates and outdated sources
- **Citation Management**: Keep track of proper citations
- **Version Control**: Update sources when new versions are available
