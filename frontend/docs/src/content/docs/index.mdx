## What is Escruta?

Escruta is an open-source platform that helps researchers, students, and professionals organize, analyze, and extract insights from their research materials using AI-powered tools.

## Core Features

<CardGrid>
  <Card title="Notebooks" icon="open-book">
    Organize research into project-based workspaces with all your sources, notes, and materials in one place.
  </Card>
  
  <Card title="AI Analysis" icon="rocket">
    Generate summaries, mind maps, study guides, and flashcards automatically from your content.
  </Card>
  
  <Card title="Smart Search" icon="magnifier">
    Find information quickly with full-text and semantic search across all your research materials.
  </Card>
  
  <Card title="Open Source" icon="github">
    100% open source with complete access to the codebase and self-hosting options.
  </Card>
</CardGrid>

## Getting Started

1. **Create a notebook** to organize your research project
2. **Add sources** by uploading documents or adding web links
3. **Take notes** with AI assistance and automatic source linking
4. **Generate materials** like summaries and study guides

## Architecture

- **Frontend**: React 19 + TypeScript + Vite + Tailwind CSS
- **Backend**: Java 17 + Spring Boot + Spring AI
- **Database**: PostgreSQL with pgvector for AI embeddings
- **Mobile**: React Native + Expo
